An AI-assisted tool for studying YouTube videos


            
Getting Started

            
    
                
  1. Paste a YouTube URL into the input bar and press Enter (or click Load).
    2. 
                
  2. The transcript and chapters will load automatically in the right panel.
    3. 
                
  3. Click Copy initialization message to copy video context, then paste it into a new Claude chat.
    4. 
                
  4. Use the Current Context section to copy your position as you discuss the video.
